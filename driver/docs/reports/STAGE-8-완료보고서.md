## 문서 핵심 포인트(사용자 친화)
- **순서 강조**: ① 서버 → ② 브라우저(Watch All Sessions ON) → ③ 드라이버.
- **새로고침 금지** 규칙을 반복 안내(웹뷰 특성 회피).
- **체험 모드 우선**: 키 없이 `npm run smoke` 로 즉시 캐릭터 확인 → 진입 장벽 최소화.
- **증상→원인→해결** 표로 흔한 오류를 모두 커버(server.json 없음, npx 안 됨, PowerShell `&&` 등).
