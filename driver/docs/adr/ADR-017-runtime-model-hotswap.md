# ADR-017: 에이전트 모델 런타임 hot-swap

- 상태: 채택(Accepted)
- 날짜: 2026-06-27
- 관련: PLAN_2 F5 (F5-1)

## 맥락

오피스 화면(F5)에서 에이전트별 모델을 선택해 **즉시 바꾸려면**, 에이전트가 실행 중에
사용 모델을 교체할 수 있어야 한다. 기존에는 모델이 생성 시 고정이었다(F4에서 폴백 전환만 가능).

## 결정

- `Agent` 인터페이스에 `getModel()` / `setModel(model)`를 추가한다.
- 내부적으로 F4의 모델 목록(`models`)·인덱스(`modelIndex`)를 재사용한다.
  - `getModel()` = 현재 모델(`models[modelIndex]`).
  - `setModel(m)`: m이 목록에 있으면 그 위치로 인덱스 이동, 없으면 현재 슬롯을 m으로 교체.
  - 변경은 **다음 tick의 LLM 호출부터** 반영된다(진행 중 호출은 건드리지 않음).
- 초기 동작은 그대로(생성 시 model 사용) → 회귀 0.

## 결과

- 외부(제어판/명령)에서 `setModel`만 호출하면 런타임에 모델이 바뀐다.
- F4의 폴백 전환과 공존한다(둘 다 같은 목록/인덱스를 다룸).
- 다음 서브스텝(F5-2~5: core 프로토콜·server 제어채널·webview UI)이 이 setter를 호출하게 된다.
