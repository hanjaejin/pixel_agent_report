# ADR-012: 공개 범위와 비밀 관리

- 상태: 채택(Accepted)
- 날짜: 2026-06-26

## 맥락

이 드라이버를 GitHub 등에 공개하려면, 누구나 클론해서 실행할 수 있어야 하고
동시에 **API 키 같은 비밀이 절대 저장소에 들어가면 안 된다.**

## 결정

1. **비밀은 환경변수/`.env` 로만**: `OPENROUTER_API_KEY` 는 `driver/.env` 또는 셸 환경변수로만 주입.
   `.env` 는 `.gitignore` 에 포함되어 커밋되지 않는다. 저장소에는 `.env.example`(키 없음)만 둔다.
2. **로그 마스킹**: 드라이버 로그는 키를 출력하지 않는다(요청 바디/헤더를 로깅하지 않음).
3. **공개 문서**: 한국어 [README.md](../../README.md) + 초보자용 [사용설명서](../사용설명서.md) +
   설계 기록(`docs/adr/`) + 테스트 로그/완료 보고서(`docs/`)를 함께 공개한다.
4. **체험 모드 우선**: 키 없이도 동작하는 `npm run smoke` 를 제공해, 외부 사용자가 키 없이
   바로 체험할 수 있게 한다(진입 장벽↓).
5. **라이선스**: 상위 pixel-agents(MIT)와 동일하게 MIT.

## 결과

- 키 유출 위험 없이 안전하게 공개 가능.
- 클론 → `npm install` → `npm run smoke` 만으로 즉시 체험 가능(키 불필요).
- 진짜 LLM 사용은 `.env` 에 키만 넣으면 됨.

## 비공개 대상

- `driver/.env`(실제 키), `node_modules/`, 빌드 산출물, `*.log` — 모두 `.gitignore` 처리.
