# 📖 아주 자세한 사용설명서 (초보자용)

안녕하세요! 이 설명서는 **컴퓨터를 처음 다루는 사람도** 따라 할 수 있게 만들었어요.
천천히 한 줄씩 따라 하면 됩니다. 어려운 말은 최대한 안 썼어요. 😊

---

## 1. 이게 무슨 프로그램이에요?

작은 인공지능(AI) 친구 3명이 **픽셀 그림 사무실**에서 일하는 모습을 보여주는 프로그램이에요.

- 김대리, 박사원, 이주임 — 이렇게 3명이 나와요.
- 각자 책상에 앉아서 📖책을 보거나, ✏️글을 쓰거나, ⚙️기계를 돌리거나, ☕쉬어요.
- 화면에서 캐릭터가 움직이고, 동시에 **글자로도** "지금 뭐 하는 중"인지 보여줘요.

> 진짜로 컴퓨터 파일을 고치는 건 아니에요. "일하는 흉내"만 내요. 그래서 **안전**해요. 👍

## 2. 시작하기 전에 (준비물 2가지)

### 준비물 ①: Node.js
컴퓨터에 **Node.js** 라는 게 깔려 있어야 해요. (이미 깔려 있을 거예요.)
확인하려면 검은 창(터미널/PowerShell)에 이렇게 칩니다:

```
node --version
```

`v20`, `v24` 처럼 숫자가 나오면 OK! 만약 "명령을 찾을 수 없다"고 나오면
https://nodejs.org 에서 Node.js 를 설치하세요.

### 준비물 ②: (선택) OpenRouter 열쇠(키)
진짜 AI로 돌리고 싶을 때만 필요해요. 처음엔 **없어도 됩니다** (체험 모드가 있어요).
나중에 필요하면 https://openrouter.ai/keys 에서 발급받으세요. (`sk-or-...` 로 시작해요.)

## 3. 검은 창(터미널) 여는 법

윈도우에서:
1. 화면 아래 **시작 버튼** 옆 돋보기(검색)를 눌러요.
2. `powershell` 이라고 칩니다.
3. **Windows PowerShell** 을 클릭하면 파란/검은 창이 떠요. 여기에 명령을 칩니다.

> 명령을 친 뒤에는 꼭 **엔터(Enter)** 키를 눌러야 실행돼요.

---

## 4. 따라 하기 (순서가 제일 중요해요!)

순서를 꼭 지켜요: **① 서버 켜기 → ② 브라우저 열기 → ③ 친구들 출근시키기**

### 🟩 1단계: 사무실 서버 켜기 (터미널 A)

검은 창을 **하나** 열고, 이 저장소가 있는 곳으로 이동해요.
(아래 `<저장소 위치>` 는 본인 폴더 경로로 바꿔요. 예: `C:\aistudy\pixelAgent_hanjaejin\pixel-agents`)

```
cd <저장소 위치>
```

**처음 딱 한 번만** 아래 두 줄을 실행해요 (시간이 조금 걸려요, 기다리세요):

```
npm install
npm run build
```

이제 서버를 켜요:

```
node dist/cli.js --port 3100
```

화면에 `Pixel Agents server running at http://127.0.0.1:3100` 같은 글이 나오면 성공! 🎉
**이 창은 끄지 말고 그대로 두세요.** (여기서 사무실이 계속 돌아가요.)

### 🟦 2단계: 브라우저로 사무실 보기

인터넷 브라우저(크롬 등)를 열고 주소창에 이렇게 치고 엔터:

```
http://localhost:3100
```

→ 책상, 의자, 화분이 있는 **빈 사무실**이 보여요.

이제 **설정**을 한 번 켜줄게요 (이걸 켜야 친구들이 잘 나타나요):
1. 화면의 **톱니바퀴(⚙️) 설정** 버튼을 눌러요.
2. **"Watch All Sessions"** 라는 스위치를 **켭니다(ON)**.
3. 설정 창을 닫아요.

> 💡 이 스위치는 "어디서 일하든 다 보여줘" 라는 뜻이에요. 켜두면 편해요.

### 🟨 3단계: 친구들(캐릭터) 출근시키기 (터미널 B)

검은 창을 **새로** 하나 더 열어요 (터미널 A는 그대로 두고요!).
드라이버 폴더로 들어가요:

```
cd <저장소 위치>\driver
```

**처음 딱 한 번만:**

```
npm install
```

이제 친구들을 출근시켜요 (키 없이 체험하기):

```
npm run smoke
```

→ 잠시 뒤 **브라우저 화면**에 김대리·박사원·이주임이 나타나서
책상으로 걸어가 일을 시작해요! 그리고 검은 창 B에는 이런 글이 흐릅니다:

```
[김대리] 🚪 출근했어요
[김대리] 📖 config.ts 파일을 살펴보고 있어요
[박사원] ✏️ main.ts 를 수정하는 중이에요
[이주임] ⚙️ npm test 명령을 실행하고 있어요
[김대리] ☕ 잠깐 쉬는 중이에요
```

축하해요! 다 됐어요. 🎉🎉

## 5. 멈추는 법

- 친구들을 퇴근시키려면: 터미널 B 에서 **Ctrl 키 + C** 를 눌러요. (3명이 퇴근해서 사라져요.)
- 사무실 자체를 끄려면: 터미널 A 에서도 **Ctrl + C**.

---

## 6. ⚠️ 꼭 기억할 규칙 한 가지

> **브라우저를 새로고침(F5)하지 마세요!**

새로고침하면 친구들이 화면에서 **사라져요**. (프로그램이 고장 난 게 아니라, 사무실 화면의 특성이에요.)
혹시 실수로 새로고침해서 사라졌다면:
1. 터미널 B 에서 **Ctrl + C** 로 멈추고,
2. 다시 `npm run smoke` 를 치면 친구들이 **다시 나타나요.**

그러니 순서는 항상: **서버 켜기 → 브라우저 열기 → (새로고침 금지) → 드라이버 실행** 이에요.

## 7. 진짜 AI로 돌리고 싶어요 (열쇠 사용)

체험 모드(`smoke`)는 정해진 행동만 반복해요. **진짜 AI**가 스스로 정하게 하려면 열쇠가 필요해요.

1. 드라이버 폴더에서 예시 파일을 복사해요:
   ```
   cp .env.example .env
   ```
2. 메모장으로 `.env` 파일을 열어요. 그리고 이 줄을 찾아 본인 열쇠를 넣어요:
   ```
   OPENROUTER_API_KEY=sk-or-여기에-본인-열쇠
   ```
3. 저장하고, 이렇게 실행해요:
   ```
   npm run start:env
   ```

이제 김대리·박사원·이주임이 **각자 다른 AI 모델**로 생각해서 움직여요.

> 🔐 **열쇠는 비밀이에요!** `.env` 파일은 다른 사람에게 주거나 인터넷에 올리지 마세요.

## 8. 안 될 때 표 (증상 → 해결)

| 이런 글/증상이 나와요 | 왜 그래요? | 이렇게 하세요 |
|----------------------|-----------|--------------|
| `server.json 을 찾을 수 없습니다` | 사무실 서버(터미널 A)가 안 켜졌어요 | 1단계로 돌아가 `node dist/cli.js --port 3100` 실행 |
| 친구들이 화면에 안 보여요 | 순서가 틀렸거나 새로고침했어요 | 서버→브라우저→드라이버 순서 지키기, 새로고침 금지 |
| 잠깐 보였다가 사라져요 | "Watch All Sessions" 가 꺼져 있어요 | 브라우저 설정(⚙️)에서 켜기 |
| `OPENROUTER_API_KEY 환경변수가 필요합니다` | 열쇠 없이 `start` 를 했어요 | 체험은 `npm run smoke`, 진짜는 `.env`에 열쇠 넣고 `npm run start:env` |
| `npx pixel-agents` 가 안 돼요 | 인터넷 배포본엔 서버가 없어요(구버전) | 이 저장소를 `npm run build` 한 뒤 `node dist/cli.js` 쓰기 |
| `'&&' 토큰은...` (PowerShell 오류) | PowerShell은 `&&` 를 못 써요 | 명령을 한 줄씩 따로 치거나 `;` 로 나눠요 |

## 9. 자주 묻는 질문 (FAQ)

**Q. 인터넷이 없어도 되나요?**
A. 체험 모드(`smoke`)는 인터넷이 거의 필요 없어요. 진짜 AI 모드는 OpenRouter에 연결해야 해서 인터넷이 필요해요.

**Q. 돈이 드나요?**
A. 체험 모드는 무료예요. 진짜 AI 모드는 OpenRouter 요금이 조금 들 수 있어요(작은 모델이라 매우 저렴).
   `PIXEL_LOOP_INTERVAL_MS` 를 크게 하면 호출이 줄어 더 아껴요.

**Q. 친구를 더 늘릴 수 있나요?**
A. `src/config.ts` 의 목록에 이름과 모델을 추가하면 돼요.

**Q. Claude(클로드)가 실행되나요?**
A. 아니요. 이 프로그램은 `claude` 를 전혀 켜지 않아요. 오직 OpenRouter(또는 체험용 가짜 AI)만 써요.
